# Linux-shell-scripts
A collection of production-ready Bash scripts for Linux system administration tasks such as service checks, automation, and operational health validation.These scripts are developed and tested on Amazon Linux 2, but are written in a portable way so they can run on other major Linux distributions with minimal or no changes.
